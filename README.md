## Краткое описание программы

**Weather Diary** — это простое и удобное десктопное приложение для ведения дневника погоды. 

Программа позволяет:
- Добавлять записи о погоде (дата, температура, описание, наличие осадков)
- Просматривать все записи в табличном виде
- Фильтровать записи по дате и по температуре
- Сохранять данные в JSON-файл
- Загружать данные из JSON-файла
- Автоматическое сохранение данных при добавлении записей

Приложение написано на **Python** с использованием библиотеки **Tkinter**.

## Технологии
- Python 3.8+
- Tkinter (встроенная библиотека)
